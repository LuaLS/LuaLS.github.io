1. Install [Ninja](https://github.com/ninja-build/ninja/wiki/Pre-built-Ninja-packages)
2. Ensure you have C++17
3. Clone the language server
   ```bash
   git clone https://github.com/LuaLS/lua-language-server
   cd lua-language-server
   ```
4. Run the make script for your OS:

<Remark type="note">
  If you are receiving a compile error similar to `cannot find -lstdc++` when
  running a make script, you may need to install `libstdc++`. On Fedora Linux or
  similar, you can do this by running the following:

```bash
dnf install libstdc++-static
```

</Remark>
